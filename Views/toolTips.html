Tool Tips:

Extension Button Functionality:

When you click the extension button, the application executes a script that scans the page's HTML. It's crucial to note that excessive clicking may result in temporary blockage from the StudentVUE site. To prevent this, refrain from rapidly clicking the button multiple times.


Dark Mode Functionality:

Extended use of light mode can cause discomfort due to excessive exposure to blue light. To alleviate this issue, we've implemented a custom dark mode theme, providing a more comfortable viewing experience.


Export Button Functionality:

The export button compiles all gathered student data on your student list page and exports it to Excel. This feature enables teachers to effortlessly create Excel spreadsheets containing comprehensive student information and grades.


Graph Button Functionality:

The graph button transforms scraped data into visual representations. Users can generate graphs for individual students or the entire class to analyze trends. Please note that the graphing functionality is limited to the data visible on the page. For comprehensive trend analysis over time, exporting the data to Excel is recommended.


GitHub Documentation:

GitHub documentation provides an in-depth explanation of the application's technical aspects and functionalities. This resource is primarily intended for developers interested in modifying the application to suit their specific requirements.
